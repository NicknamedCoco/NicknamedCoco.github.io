Hexo安装

Hexo+GithubPages的方式配置静态站点还是挺简单的，适合我这种懒人，在本页面记录下，方便以后查看及修改。

hexo官网：https://hexo.io/zh-cn/

  1. 安装hexo需要先安装Node.js和Git

    Node.js官网：https://nodejs.org/zh-cn/

    Node.js安装教程：https://blog.csdn.net/antma/article/details/86104068

    Git官网：https://git-scm.com/

    Git安装教程：https://www.jianshu.com/p/414ccd423efc

  2. 此时安装hexo命令： npm install -g hexo-cli

    但执行之前最好先切换成淘宝源，以防卡住npm config set registry https://registry.npm.taobao.org

更多的信息直接查看Hexo文档就行，这里不过多阐述。

Git的配置及githubPages的配置等可以查看下面的Hexo配置链接

https://zhuanlan.zhihu.com/p/60578464

https://www.jianshu.com/p/51cc016a1933

https://blog.csdn.net/sihai12345/article/details/79306483

Hexo主题更换： https://www.jianshu.com/p/9f794db64f42

Hexo基础命令

​ hexo clean —清除站点缓存

​ hexo g —生成站点信息

​ hexo s —hexo的测试页面

​ hexo d —同步githubPage,上传数据

​ hexo new “文章名” —_posts下创建一篇markdown文章

​ hexo new page 目录名 —在source下创建文件夹
